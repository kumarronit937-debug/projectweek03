# projectweek03

Styled Portfolio Website Documentation

1. Project Overview

This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript.

The website contains:

- Header and navigation menu
- About section
- Skills section
- Contact form
- Profile image
- External links
- Dark mode functionality

Objective

The objective of this project is to practice frontend web development concepts including webpage structure, styling, responsive design, and JavaScript interactivity.

---

2. Setup Instructions

Step 1: Download the Project

Download or clone the project repository from GitHub.

Step 2: Open in VS Code

Open the project folder using Visual Studio Code.

Step 3: Run the Project

Open the "index.html" file in a web browser.

No additional installation or software setup is required.
